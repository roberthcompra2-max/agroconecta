# 🌱 AgroConecta - Campo e Cidade Conectados

O **AgroConecta** é uma plataforma inovadora desenvolvida para aproximar produtores rurais (agricultores) de estabelecimentos comerciais (supermercados e mercados). O foco é facilitar o escoamento da produção local com transparência e agilidade.

---

## 🚀 Funcionalidades Principais

### 👨‍🌾 Para o Agricultor
- **Cadastro Simples:** Perfil dedicado com ícone de trator para identificação visual rápida.
- **Postagem de Produtos:** Upload de fotos, definição de preço, cidade e estado (UF).
- **Gestão de Anúncios:** Visualização em tempo real dos itens publicados.

### 🛒 Para o Mercado (Comprador)
- **Barra de Pesquisa Inteligente:** Filtro avançado por nome do produto, cidade ou estado.
- **Indicadores Estáticos:** Dólar Comercial e Clima (Açailândia-MA) visíveis no topo para auxiliar na tomada de decisão de preços.
- **Negociação Direta:** Botão integrado para abrir conversa no WhatsApp do produtor.

### ⚙️ Tecnologia Utilizada
- **Frontend:** HTML5, CSS3 (Design Responsivo) e FontAwesome.
- **Backend/Database:** Firebase Firestore (Banco de dados NoSQL em tempo real).
- **APIs Externas:** Consumo de API de economia para cotação do Dólar em tempo real.
